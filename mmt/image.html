The Image Principle

How It Works

The image principle is short and sweet: including an image of the instructor does not necessarily improve learning.

Yes, there can be a benefit from the sense of the presence of the instructor, but it might be outweighed by the extra cognitive load of processing a human face. (Remember that cognitive load refers to the amount of memory resources used for a task -- we always want to reduce extraneous cognitive load so that students can focus on the task at hand.)
